===========================
helpdesk_portal_ticket_list
===========================

Customize ticket portal view list

Installation
============

Use Odoo normal module installation procedure to install
``helpdesk_portal_ticket_list``.


Description
============
In ticket portal view list
- expand "name" column
- fix display bug on "Stage" column
